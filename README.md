## Description

Đây là một dự án nhỏ về việc quản lý công việc
+ Tạo nhóm
+ Thêm người vào nhóm
+ Tạo công việc
+ Thêm mới công việc

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# install package
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
